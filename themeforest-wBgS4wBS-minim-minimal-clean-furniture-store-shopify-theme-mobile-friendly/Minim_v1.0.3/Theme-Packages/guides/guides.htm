  • Get Started

    Minim is a highly flexible, Multi Store Responsive Shopify Theme. It's built on Bootstrap 3 and looks great on Desktops, Laptops and Mobile devices. With unique cool designs, modern, stylish, intuitive assistant and 6 Unique Home layouts, Minim is a good choice to build your online store. The Minim shopify theme supports 3 view modes for product listing page: sidebar , all the views are lean, show info that user wants.

    The theme includes all shop pages: Product listing, product detail, add to cart, checkout, quick view ... with customized layout and style to make it look incredible. The product filtering system is powered with the integration of modules: Categories list, Filter based on product attribute, New products, Related products... to help user find out their best fit items at a glance. The Shopify theme is easy to customize with the Theme Customizer tool that let you change settings for most of the pages, sections in the site based on predefined parameters.

    The multi-purpose Multi Store Responsive Shopify Theme is SEO optimized that makes your site friendly with Search Engines, CSS and JS compression helps to improve your site performance a lot.

  • Variant Image Change

  • Variant Image Change

    With Minim, you can showing more than one product images with color, please follow this instruction.

    1. The first you need order again your variant color as group.

    2. From the images of your product, You should sort the image to each groups and according to order. Example: Your color are Dusty Pink -> Green -> Taupe -> Grey.

    So you need organize order image as: all product Pink -> all product Green -> all product Taupe -> all product Grey.

    3. Final, from your Variant, you need choose product image for first variant of color.

    4. Result: When you click on color then image will show corresponding to that color.
